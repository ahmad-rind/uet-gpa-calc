## 📌 About

A responsive GPA calculator built specifically for **UET Taxila** students.
Calculate your semester GPA and cumulative CGPA quickly with a clean UI.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📊 GPA Calculator | Calculate semester GPA by entering grades and credit hours |
| 🔄 CGPA Tracker | Cumulative GPA across multiple semesters |
| 📱 Responsive Design | Works on mobile and desktop |
| ⚡ Instant Results | No page reload, real-time calculation |
